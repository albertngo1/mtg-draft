The **consolidated, agent-primary** guide — synthesized across all 10 episodes into one dense,

## ⚠ Recency rule

These episodes span **2026-04-13 (prerelease) → 2026-06-01 (retrospective)** and the format evolved
a lot over that window. **On any conflict, the newest video wins.** Prerelease / early-access takes
(Apr 13–20) are *predictions* and the weakest evidence; the **2026-06-01 "School's Out For Summer"
retrospective is the most authoritative single source.** The guide already bakes this in (supersessions
are marked inline). Example: Witherbloom was *predicted* #1 on 04-13 but finished **dead last** per the
06-01 retro.

## SOS quick reference (the five colleges)

| College | Colors | Theme |
|---------|--------|-------|
| Silverquill | WB | aggro / +1+1 counters |
| Prismari | UR | spells / magecraft |
| Witherbloom | BG | lifegain / pests / grind |
| Lorehold | RW | tokens / go-wide / artifacts |
| Quandrix | GU | ramp / big creatures / fractals |

**Recurring meta read:** "soup" (greedy multicolor converge/value/control) outperforms the aggro
decks; midrange-with-interaction is strong; white and blue are the best colors. See the guide for
the full tier list and the 17Lands-vs-hosts disagreement on soup.
