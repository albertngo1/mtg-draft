# Lords of Limited — Murders at Karlov Manor (MKM) Notes

Agent-facing draft-strategy reference for **Murders at Karlov Manor** Limited, distilled from the
[Lords of Limited](https://www.youtube.com/@LordsofLimited/videos) YouTube channel.

MKM originally released Feb 2024; these notes are from that window. The cards are **identical** in the
2026 MTG Arena Quick Draft re-run (**June 8–16, 2026**), so the strategy applies directly.

## → Start here: [`MKM-draft-guide.md`](../MKM-draft-guide.md)

The **consolidated, agent-primary** guide — synthesized across all 6 episodes: recency rule, a
**2026 Quick-Draft data caveat** (see below), TL;DR meta read, 10-guild tier table, MKM format
principles (ward-2 Disguise, Clues, Collect Evidence, Cases, Suspect), a **card-notes table**, signals,
and supersession/disagreement sections. Hold it in context for a whole draft. Treat as expert/theory
opinion — **17Lands GIH WR stays primary.**

## ⚠ Data caveat for the 2026 re-run

Before/early in the Quick Draft window, 17Lands **QuickDraft** data for MKM may be empty (0 games).
Until it populates, proxy with MKM's original **PremierDraft** window (`--fmt PremierDraft`,
`start_date=2024-02-01`). For the tool: `./mtg-draft.sh pull --set MKM` once QD data is live.

## MKM quick reference (the 10 guilds)

| Guild | Colors | Theme |
|-------|--------|-------|
| Azorius | WU | Detectives / tempo / Clues |
| Dimir | UB | mill / Surveil / Cases |
| Rakdos | BR | sacrifice / Suspect |
| Gruul | RG | +1/+1 / Suspect aggro |
| Selesnya | GW | go-wide / Clue tokens |
| Orzhov | WB | aristocrats / lifegain / midrange-aggro |
| Izzet | UR | artifacts / spells / Clues |
| Golgari | BG | graveyard / Collect Evidence value |
| Boros | RW | aggro / Disguise + combat |
| Simic | GU | Investigate / clue ramp / value |

**Retro meta read:** White best color, **Boros (RW) the top archetype**, **Simic (GU) #2**, Azorius #3;
**Dimir (UB) worst**. Ward-2 Disguise warps removal evaluation. See the guide for the full tier list,
card notes, and early-prediction-vs-retro reversals.
